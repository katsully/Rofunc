# FBX SDK installer

install FBX SDK, install FBX Python SDK,and Python bindings

  Define the FBXSDK_ROOT environment variable to the location of the
     installed FBX SDK 

cd to python bindings (where the toml file lives)
activate conda env
python -m pip install --upgrade pip setuptools wheel

Install Sip version 6.6.2 (newer version have template issue)
    python -m pip install --force-reinstall -v "sip==6.6.2"
Build and Install command:
    python -m pip install .
